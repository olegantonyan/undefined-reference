I'm a software engineer working since the 2010s. I started with embedded development for 8- and 32-bit microcontrollers in `C` and `C++`, moved through mobile development, and now work mostly with `Ruby`, `Rails`, and data engineering.

This blog collects short, practical notes on programming — mostly Ruby and Rails, plus the occasional detour into Rust, C/C++, software design, and debugging. I write up the things I wish I'd found in someone else's blog: small patterns, gotchas, and techniques that saved me time.

Topics you'll find here: Ruby and Rails idioms, service objects and design patterns, metaprogramming, debugging, and integrating native code with Ruby.

### Projects

- <a href="https://omnipackage.org/" target="_blank" rel="noopener">OmniPackage</a> — build native Linux packages (RPM, DEB, pacman) for multiple distros and host your own repository.
- <a href="https://mpz-player.org/" target="_blank" rel="noopener">mpz</a> — a cross-platform music player for large local collections that uses your folder structure as the library (C++/Qt).
